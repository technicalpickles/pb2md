I generally recommend some variation ofchoose one of these:
  • Just to be sure I'm clear...
  • I want to make sure I know the best way to help...
  • Tell me more...
+ one of these:
  • What's the underlying problem you're trying to solve?
  • If we had that completed already, what's the benefit you'd expect to get?
  • What does your ideal outcome / end goal look like?
Anytime we ask 'why?' it can sound like we're being defensive or reluctant to help, so it can help to add something like:
  • "that way we can figure out the best options together"
  • "I want to make sure I'm answering the most relevant questions for you
etc.(some key phrases in slides for this talk for the recent LEAP cohort on how to effectively ask for help)
